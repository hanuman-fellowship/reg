Adding an Event with a date range the Name can contain:

No PR
No PRs
no pr
No PR indoors
No PR indoor

No Meal
no meals

No me
No MEs

And combinations:

No PR ME Meals indoors

The key is the word 'no' surrounded by space (or the beginning of the line)

Me (mountain experience) is distinguished from Meals

ME end dates vs PR end dates
PR end dates are -1 - yes

After an Event is added it is checked and all values are updated on mountmadonna.org

There are also the Strings pr_max and me_max AND now there are PR/ME Date Ranges that supercede these.
