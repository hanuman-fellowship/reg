Adding an Event with a date range the Name can contain:

No PR
No PR indoor
No Meal
No Lunch
No ME

And combinations:

No PR ME Meal indoor


The key is the word 'No' surrounded by space (or the beginning of the line)

ME (mountain experience) is distinguished from Meal

ME end dates vs PR end dates
PR end dates are -1 - yes

Note that we ALSO have:

Yes ME

Both No ME and Yes ME interact with the String me_dow. See cgi-bin/regME on mountmadonna.org

The words ME, PR, and Meal can be in mixed/lower case and have an optional 's' afterwards. After these kind of Events are added values are updated on mountmadonna.org.

There are the Strings pr_max and me_max AND Date Ranges ME/PR that supercede these.
