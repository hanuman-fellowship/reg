Searching for Registrations

There are two search fields - one for the Registrant and one for the Program.

In the Registrant field you can put a prefix of the last name and, if you wish, a prefix of the first name.

In the Program field put a prefix of the program name. If you want to search for people in the current program just leave this field blank.

For example:

    Registant: ar c
    Program: av
This searches for people whose last name begins with ‘ar’ and whose first name begins with ‘c’ - in programs whose name begins with ‘av’.

In both fields you can include a star ‘*’ - for wildcard matching.

    Registant: *son
    Program:
The above will look for people in the current program with ‘son’ in their last name somewhere.
