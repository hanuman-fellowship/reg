ProgramDoc To Help for Developers

A document that is attached to a program. It is uploaded to the web and a link is made to it on the program's web page.

Table: program_doc

id
unique id
program_id
foreign key to program
suffix
the file extension of the document (e.g. jpg or pdf)
title
The description of the document - used as a link target on the web page.
