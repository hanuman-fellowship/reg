RequestedMMIPayment To Schema

Payments for MMI programs can be sent to the registrant so they can pay online via authorize.net. When the payment is made at authorize.net it is brought in automatically by Reg.

Table: req_mmi_payment

amount
dollar amount
code
a generated code sent to the person to identify the payment request
for_what
Tuition, Meals and Lodging, STRF, Recordings, Other
id
unique id
note
the reason for the charge
person_id
foreign key to person
reg_id
foreign key to registration
the_date
the date the request was sent

Relations

personpersonbelongs_to
registrationregistrationbelongs_to
