MMI Online Registration

Some MMI Courses are made available to the public. These courses are not just for students who are working towards a formal Diploma, Certificate, or Masters accredited degree from MMI. These courses are advertised widely and there are web pages at mountmadonnainstitute.org to allow anyone to register for the course online. This document outlines the steps needed to make the above happen.
  1. Define the public course - its curriculum and its teacher.
  2. Create a program in Reg for this course - you specify (at least) these things:
    • Name
    • Start and End dates
    • Housing Cost (Meals/Lodging tab)
    • On the Misc tab:
      • Category - Normal
      • Sponsoring Organization - MMI
      • Level - Public Course
    • Tuition (Finance tab)
    • Title (Web tab)
    • Check 'Web Ready' (Web tab)
  3. At mountmadonnainstitute.org create a web page that describes the course completely. To make it easy to find, make links to this page from various places on the site. Somewhere on the course web page there should be a 'Register Here' link like so: 
        >Register Here
    
    Here is the HTML for the link:

    <a href=http://www.mountmadonnainstitute.org/cgi-bin/reg1?dir=courses&id=3661>&gt;Register Here</a>

    The purple href above is the same for all public courses. The red 3661 above is the ID of the course's program in Reg.

    How to easily get the program ID and the href? See the next step.

  4. When looking at the Programs in Reg you will see the command "MMI Publish". When you choose it, Reg will gather all future MMI public courses and 'publish' them on the mountmadonnainstitute.org site. In this context, 'publish' means that information is made available to that site to facilitate online registration. It does not alter any web pages on the site.
