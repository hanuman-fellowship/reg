The first row of the spreadsheet needs to be a header row.
These column headers are required to be present (but not necessarily in this order):
    first, last, address, city, state, zip, country, phone, email
These are optional:
    address2, company
If a required header is missing the upload will be abandoned with an error message.

The letter case of the header string is ignored so

    First, first, or FIRST
are all okay.

We only look for a substring that matches so

    Zipcode
will be accepted for 'zip'.
