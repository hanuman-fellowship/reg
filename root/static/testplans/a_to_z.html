Here we test the new registration program by creating
a program anew, publishing it to the web, registering online,
sending confirmation letters, registering manually,
cancelling, housing, accepting payments when the people arrive,
printing meal lists, housekeeping reports, late notices,
ceus, participant lists, summaries, etc.  i.e. everything from A to Z.
Oh and then we have rentals.

Testing now is crucial.  The old reg evolved over a long
period of time.  This is _new_ software that has not been
subjected to the fire/kiln of time.  There are weaknesses
and frailties that need to be found and remedied.
This is _much_ better done BEFORE we 'go live'.
This IS work and takes time.  It is important.
I will insist that it is done fully before we 'go live'.
Fully by more than one person even.

Create a New Program
    make a new cancellation policy
    make a new housing cost
        have values that are easily distinguished from the default
    add the leader as a person (include phone, email)
    make them a leader (include bio, phone, email, web address)
        also upload a picture of the person.
    add a new affiliation for this program
    create the program
        use the 4 tabs
        set the date a month in advance of 'today'.
            make it 4 or 5 days long
        no extra days
        set tuition and online deposit
            to an unusual amount so you'll recognize it
        set footnotes to * (CEU for nurses)
        assign the housing cost and the cancellation policy
            you created above.
        some things to set after creation choosing Edit:
            set a few days for lunch
            assign some housing clusters
            assign the leader you made above
                note they are automatically registered
                but that they need to be housed.
            assign meeting and breakout places
                notice the effect of the Max of the program
    
Publish to the Web
    Also publish the pictures.

Register Online
    see the web page for the program
    register online
        enter a recognizable person (a newly created name)
        with your email address (so you can recieve the confirmation letter)
        make some mistakes and get the errors
    choose housing options
    enter some special request
    use credit card of 411111111111 (4 followed by 1's to fill the field)
        expire 0909
    receive the authorize.net notification email

Get the Online Registrations
    go to the Programs link and choose "Grab New"
        to get the just entered online registration
    select it.
    see the special requests, housing options
        default housing type is 1st choice
    select a house
        look carefully at the list.
        see its order and the markings (RFOCS)
        verify.
    finish the registration
    choose "Send" to send off the confirmation letter
    receive it in your email.

    verify that the affiliation for the program
        got to the person's demographic screen.

Look at the Calendar and DailyPic and verify that
    everything looks good and as expected.
    Repeat this verification as you follow the rest of this testplan.
    Click on the program boxes and see that you go to that program.
